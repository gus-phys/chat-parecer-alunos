# Estrutura dos arquivos
```bash
\chat-parecer-alunos

│   ├── preprocessing       # Dados processados
│   └── raw                 # Dados brutos
├── notebooks               # Jupyter notebooks
├── src                     # Códigos da aplicação

│   ├── models              # Códigos de modelos
│   ├── preprocessing       # Códigos de processamento de dados
│   └── utils               # Códigos auxiliares
├── tests                   # Testes da aplicação
├── README.md               # Documentação principal do projeto
├── requirements.txt        # Requisitos do ambiente
└── setup.py                # Configuração do ambiente
```
